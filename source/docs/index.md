Autossential.Activities is a comprehensive set of activities designed to simplify and accelerate everyday RPA development tasks in <a href="https://www.uipath.com" alt="UiPath" target="_blank">UiPath</a>.
The activities are organized into categories such as Data, Files, Programming, System, and Workflow, making it easy to find the right tool for the job.

The version 4.x was completed redesigned aiming exclusively in the compatibility with Windows and Cross-Platform UiPath projects. Windows-Legacy projects are no longer supported.

The package is available through NuGet (preview and stable releases) and the eventually in the UiPath Marketplace (stable releases only).
