# Introduction

The Autossential.Activities is the main package containing 40+ activities.

Provides a collection of activities to work with DataTables, Collections, Files, Programming, Workflows, Security and Diagnostics.

The package is available on NuGet (preview and stable versions) and also UiPath Marketplace (stable versions only).

!!! warning "About version 3.x"
    While version 3.x remains compatible with Windows projects in UiPath, we strongly recommend upgrading to version 4.x.
    Although version 3.x is stable, it will no longer receive maintenance or updates.

    For cases where support for Windows-Legacy projects in UiPath is still required, you may continue using version 3.x.
